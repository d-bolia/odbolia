# odbolia — Personal Portfolio & Resume Website

## Project Overview

Personal portfolio and resume website for **Desmond Bolia**, an Electrical Engineer (B.S. EE, UC Irvine — semiconductor and optoelectronics track) with prior industry experience as a **Project Engineer at Helix Electrical, Inc.**

## Tech Stack

- **Framework:** Next.js (App Router)
- **UI:** React, Tailwind CSS
- **3D / Animation:** Three.js, React Three Fiber (R3F)

## Design Vision

- **Palette:** Dark background, modern and minimal with a technical/engineering aesthetic
- **Hero:** Animated wireframe sphere with an amorphous, fluid surface resembling the surface of the sun in wireframe form — built with Three.js/R3F, reacts to cursor movement
- **Background:** Animated net/particle mesh (reference: 109ichiki.com) — dark, flowing, techy
- **Layout:** Vertical scroll, clean spacing
- **Motion themes:** Circular and orbital motion throughout
- **Anti-patterns to avoid:**
  - No Inter font
  - No purple gradients on white
  - No generic card grid layouts
  - No AI-slop aesthetics
- **Typography:** Should feel engineered, purposeful, not generic — consider monospace or structured sans-serifs that reinforce a technical identity

## Page Sections (in order)

1. **Hero** — Wireframe animated sphere + name + title
2. **About / Profile** — Short personal bio
3. **Portfolio / Projects** — Showcase of engineering projects
4. **Resume** — Education, experience, skills
5. **Contact** — LinkedIn and GitHub links
